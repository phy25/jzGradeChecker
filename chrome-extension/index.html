Chrome / Firefox 扩展安装 打包 Chrome 便携版下载 扩展介绍

上面可以快速安装，如需要自己选择安装方式，看下面。

版本 0.8.3（更新于 2016/8/1，更新日志）

安装完成后，打开学校查成绩的网页即可开始使用。打包 Chrome 版直接打开“运行 jzGradeChecker.bat”即可。

多核浏览器（360 安全浏览器、360 极速浏览器等）访问学校查成绩页面时可能会自动切换到“兼容模式”（IE 模式），这种情况下扩展无法改写页面。请手工切换到“极速模式”，扩展才能正常工作。

使用自带浏览器的打包版本

百度云下载页面（Windows 版，约 50M，扩展版本见文件名）

本版本自带了 Google Chrome 浏览器。只需要下载一个 exe 文件，自动解压后，运行 jzGradeChecker.bat 即可马上使用扩展功能。
